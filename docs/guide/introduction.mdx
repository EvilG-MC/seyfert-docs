# Seyfert

Seyfert is a powerful library that focuses on Discord and its bot API. It offers support for scalable applications and provides unique utilities.

## Installation

### Prerequisites

- NodeJS installed (18 or higher)
- TypeScript installed (4.9 or higher)
- npm installed (or similar NodeJS package manager)
